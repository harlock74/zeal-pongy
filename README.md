# zeal-pongy (Still working in progess)

Pongy is a Pong inspired game.

Binaries
zeal-pongy.bin - this build should work on both emulator and hardware

Controls
SNES D-Pad, WS, Arrows for movement (up/down)
SNES B to Launch the ball
SNES Start or Enter to Pause/Resume
SNES Select or Single Quote (') to Quit
